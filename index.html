The Builder

Hyper Text Markup Language

HTML or HyperText Markup Language, helps us structure the web page. It is called a markup language as it marks up each part of its content defining what it is and does. HTML basically helps us build and structure the content of our webpage and it does so with a series of elements that altogether different parts of our content are made to appear and behave in a certain way.

The Artist

Cascading Style System

CSS or Cascading Style System helps us style our webpage. HTML without CSS would be just words in a blank page. What CSS does is help HTML look pretty and cool. It can give us a background color, borders with colors on pictures and so on. CSS selectively helps us style our HTML elements. Unlike HTML it is not a markup language but a style sheet language.

The Wizard

JavaScript

JS or JavaScript helps us add a sort of interaction between the user and our website. JS is a dynamic programming language which adds interactions on stuff that are made on HTML. Frontend

When building a website we need to assemble all of the files we want to use into a structure on our local PC. We need to ensure that all the files work perfectly with each other and look good before uploading it live to a server. In order to achieve this, we will need to use our languages. These guys work together and if one of them misses from our webpage then we probably have created something pretty unpleasant for the eye or barely functional.
